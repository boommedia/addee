## 📱 Social Media Autopilot — Publish to 4 Platforms with 1 Click

**What:** Publish blog posts directly to Google Business, LinkedIn, Medium, and Dev.to
**Cost:** $15/mo add-on

## 🚀 Quick Start

1. Go to **Billing** → Add the **Social Media Autopilot** add-on ($15/mo)
2. Open any blog post → scroll to **Social Media Autopilot** section
3. Click **"Connect [Platform]"** for each platform you want to use
4. Once connected → **Click "Publish Now"** to go live

---

## 🌍 Platforms

### Google Business Profile
- **For:** Local businesses, service providers, stores
- **Setup:** Click "Connect" → Sign in with your Google account
- **Publish:** Add optional image → Click "Publish Now"
- **Reaches:** Google Search, Google Maps, local pack
- **💡 Tip:** Short text (100-200 chars) + good image = 80% more engagement

### LinkedIn
- **For:** B2B companies, agencies, professionals
- **Setup:** Click "Connect" → Sign in with LinkedIn
- **Publish:** Add optional image → Click "Publish Now"
- **Reaches:** LinkedIn feed, your followers
- **💡 Tip:** Add a question to drive engagement comments

### Medium
- **For:** Long-form articles, thought leadership, tutorials
- **Setup:** Go to **Settings** → **Integrations** → Add your Medium API token
  - Get token: `medium.com/me/settings/security` → Integration tokens → Create new
- **Publish:** Click "Publish to Medium"
- **Reaches:** Medium's millions of readers
- **💡 Tip:** Medium readers want depth — full articles do best

### Dev.to
- **For:** Technical blogs, tutorials, developer audience
- **Setup:** Go to **Settings** → **Integrations** → Add your Dev.to API key
  - Get key: `dev.to/settings/account` → DEV Community API Keys → Generate new
- **Publish:** Click "Publish to Dev.to"
- **Reaches:** Dev.to's 2M+ developers
- **💡 Tip:** Use 4-5 relevant tags (#javascript #tutorial #react etc.)

## 🔧 Troubleshooting

**Popup blocked?**
- Look for popup notification in browser address bar
- Allow popups for bloggy.online

**Connection not working?**
- Try incognito/private window
- Clear browser cache
- Make sure you're using the right account

**Post didn't appear?**
- Wait 60 seconds (sometimes there's a delay)
- Log into the platform directly to check
- Try publishing again

## ❓ FAQ

**Q: Can I publish to some platforms but not all?**
A: Yes! Only connect what you use.

**Q: What if my plan limit is 20 posts/month — does Social Media Autopilot count against it?**
A: No. Social Media Autopilot posts don't count against your plan's post limit.

**Q: Can team members use it?**
A: Yes! They'll see the publish buttons. Whoever sets up the connections is the account owner.

**Q: What happens when I cancel the add-on?**
A: Posts stay live on the platforms. If you re-enable, your connections are still there.
