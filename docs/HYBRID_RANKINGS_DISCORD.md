## 🎯 New: Hybrid Rankings Dashboard

**What:** See keyword rankings from Google Search Console + DataForSEO (all users) + LocalFalcon grid data (Agency+ only)  
**Live now:** Analytics → Rankings tab  
**Cost:** Basic rankings included · LocalFalcon exclusive to Agency+ plans

## 📊 What You Get at Each Tier

### ✅ All Users
- **Keyword Rankings:** GSC clicks, impressions, CTR + DataForSEO positions
- **KPI Cards:** Total keywords tracked, top 3, top 10, average position
- **Keyword Table:** Position, search volume, clicks, impressions, CTR
- **Source badges:** Shows if data came from GSC, DataForSEO, or both

### 🔒 Agency+ Only
- **LocalFalcon Grid:** ARP, SoLV, ATRP metrics across mapped grids
- **Competitive breakdown:** Performance by location & keyword
- **Historical trends:** Track movement over time
- **Heat maps:** Visual ranking patterns

## 🚀 Quick Start

1. Go to **Analytics** → **Rankings** tab
2. **Connect Google Search Console** (in Settings) to see clicks & impressions
3. Basic rankings appear immediately
4. **For LocalFalcon data:** Upgrade to Agency ($149/mo) or Agency Max ($299/mo)

## 🔧 Troubleshooting

**No data showing?**
- Make sure GSC is connected in Settings
- DataForSEO rankings update weekly
- LocalFalcon requires Agency plan

**LocalFalcon not visible?**
- Check your subscription in Billing
- Must be Agency or Agency Max tier
- Contact support if you see an error

## ❓ FAQ

**Q: Do basic rankings cost extra?**  
A: No! Keyword rankings from GSC + DataForSEO are included for all users.

**Q: What's the difference between ARP and position?**  
A: ARP = LocalFalcon grid average · Position = DataForSEO/GSC rank at specific search query.

**Q: Can I see historical ranking data?**  
A: Yes! DataForSEO and LocalFalcon both track history. Scroll through past scans.

**Q: How often do rankings update?**  
A: GSC = daily · DataForSEO = weekly · LocalFalcon = on-demand (run scans anytime).
