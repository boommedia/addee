💡

Annual pricing: Offer 2 months free (pay 10, get 12) — approximately 17% discount. This dramatically improves cash flow and reduces churn. Target 40%+ of customers on annual at steady state. Also consider a 7-day free trial (no credit card required) — agencies buy after seeing it work on their actual clients.

⚠️

Overage pricing matters: Set overage rates that are profitable but not punishing. The $0.99/blog overage on Starter still gives you ~70% margin. Agencies that hit limits regularly should be upsold to the next tier — track this and trigger an upgrade email automatically.

Feature Architecture
What the Agency SaaS Adds (vs Internal Tool)
The core blog engine is the same as your internal Boom Media tool. The SaaS version adds the multi-tenant layer, billing, white-label, and everything that makes it resellable.
🏗️

Multi-Tenant Architecture New

  • Row-level security in Supabase — each agency only sees their data
  • Agency → Client → Blog post hierarchy in DB schema
  • Isolated generation queues per agency
  • Per-tenant usage metering and overage tracking
  • Soft-delete architecture for cancellations
💳

Billing & Subscription System New

  • Stripe Billing — all 4 plan tiers + annual toggle
  • Metered overage billing (per-blog overages auto-charged)
  • Self-serve plan upgrades and downgrades
  • Billing portal (Stripe Customer Portal)
  • Invoice generation for Enterprise accounts
  • Churn webhooks → trigger win-back sequences
🏷️

White-Label Dashboard (Pro+) New

  • Agency uploads their own logo and brand colors
  • Custom subdomain: app.theiragency.com
  • Branded client-facing reports (PDF export)
  • Remove all "Powered by" branding on Pro+
  • Custom email sender domain for notifications
🔑

Agency Admin Console New

  • Manage all client workspaces from one view
  • Add/remove team members with role permissions
  • View usage across all clients in real time
  • Bulk actions (assign brand voice, schedule posts)
  • Client access toggle (give client limited read view)
🔌

API + Webhooks (Pro+) New

  • REST API for blog generation programmatically
  • Webhook events: post.created, post.published, quota.warning
  • Zapier / Make integration via webhooks
  • API key management in dashboard
  • Rate limiting and API usage logs
🚀

Onboarding & Growth Engine New

  • Guided setup wizard (connect first client + generate first post)
  • In-app onboarding checklist (drives activation)
  • Automated upgrade nudge emails (Resend + Loops)
  • Referral program — agencies refer agencies
  • NPS survey at 30/90/180 days
📥

Content Ingestion (Same Core)

  • Text input / prompt (first-class source)
  • YouTube URL → transcript
  • Audio / podcast → AssemblyAI transcription
  • Webpage scraping → blog repurpose
  • PDF / Doc upload
  • TikTok / Vimeo → yt-dlp
📊

Analytics (Enhanced for Agencies)

  • Per-client blog performance dashboard
  • Aggregate view across all clients
  • SEO ranking trend tracking
  • White-label PDF reports clients can send to their clients
  • Usage alerts when approaching plan limits

Customer Support
Support Requirements by Plan Tier
Agencies are professionals — they expect fast, knowledgeable responses. Set clear SLAs per tier and staff to meet them. Unmet SLAs are the #1 churn driver for agency SaaS.
Starter

Self-Serve Support

SLA: 48-hour response
  • Email support only (Crisp inbox)
  • Knowledge base / help docs (Notion or GitBook)
  • In-app tooltip guidance and onboarding wizard
  • Community Slack or Discord (async)
  • Video walkthrough library (Loom embeds)
  • No live chat, no calls
Growth

Standard Support

SLA: 24-hour response
  • Email + in-app chat support
  • Full knowledge base access
  • Community access
  • Monthly office hours (group Zoom)
  • Bug reports prioritized within 48 hrs
  • Onboarding guide (self-paced)
Pro

Priority Support

SLA: 4-hour response
  • Priority email + live chat
  • 1 onboarding call (60 min) included
  • Shared Slack channel (optional)
  • Bug reports escalated immediately
  • Quarterly check-in call
  • Feature requests tracked with feedback
  • White-label setup assistance
Enterprise

Dedicated Success Manager

SLA: 1-hour response
  • Named Customer Success Manager (CSM)
  • Direct Slack channel with your team
  • Monthly strategy + review calls
  • Custom onboarding plan (2–3 sessions)
  • Guaranteed bug fix SLAs (P1: 4hr, P2: 24hr)
  • Early access to new features
  • Quarterly business reviews (QBR)
🎯

Support tooling to set up: Crisp for chat + email ($25/mo to start) → upgrade to Intercom at $50K+ MRR. Use Notion for help docs initially. Add a status page (Instatus, $20/mo) from day one — Enterprise customers will ask for it. Track response times in your support tool from day one so you can prove SLA compliance.

Go-To-Market
Getting Your First 5–10 Agency Customers (Beyond Paid Ads)
Paid ads get you volume, but your first customers come from trust. Use these organic channels to amplify your signal and get quick wins.
1. Start with Boom Media

Use your internal tool first. Build a case study: "We generated X blogs for Y clients in Z weeks." That's your first sales asset — agencies trust agencies.

2. 5 Free Beta Agencies

Give 5 hand-picked agencies 90 days free in exchange for feedback + a testimonial. Pick ones with 10–20 clients for maximum signal. DM them directly on LinkedIn.

3. Agency Facebook Groups

Groups like "Agency Owner Community", "SMMA Owners", "SEO Professionals" have 50K–200K members. Share results, not ads. Contribute value first.

4. LinkedIn Outbound

Filter: "Content Marketing Agency" + "Founder/Director" + 10–200 employees. Personalized video DM showing their competitor's content vs what your tool would generate.

5. AppSumo / Lifetime Deal

Consider an AppSumo launch at $199 lifetime (limited to 1,000 codes). Gets you cash, reviews, and word-of-mouth. Do this only after the product is solid.

6. Referral Program

Give agencies 20% recurring commission for referrals. Agency owners talk to each other constantly — one Growth customer can send you 3–5 more. Track with Rewardful.
