Create the Server

Get the server set up with your branding before anything else.

  • 1
    Open Discord → click the + icon in the left sidebar (Add a Server)
  • 2
    Select "Create My Own" → then "For a club or community"
  • 3
    Server name: Bloggy Community
  • 4
    Click the server icon area → upload the Bloggy logo (square PNG, at least 512×512)
  • 5
    Click Create
  • 6
    Go to Server Settings → Overview → set Server Description: "The official community for Bloggy — AI blog automation for agencies. Get support, share wins, and connect with other agency owners."
  • 7
    Under Server Settings → Community → Enable Community features (gives you welcome screen, rules, announcements)

💡 Tip

Go to Server Settings → Invites → Create Invite → Never Expires and save that link. You'll need it for the Bloggy website later.

2

Create Roles

Set up roles before channels — you'll need them for permissions. Do these in order.

  • →
    Go to Server Settings → Roles → Create Role for each role below
  • →
    For each role: set the name, pick the color, enable "Display role members separately" so they show in the sidebar
  • →
    Drag roles in this order from top to bottom: Admin, Pro Member, Free Member, Visitor
🛡️ Admin
Color: Red
You only. Full permissions. Keep this at the top of the role list.
⭐ Pro Member
Color: Gold
Paying Bloggy subscribers. Gets access to exclusive Pro channels.
🌱 Free Member
Color: Green
Free account users. Self-assigned via reaction role.
👀 Visitor
Color: Gray
Default for anyone who joins. Limited read-only access.

⭐ Pro Member Upgrades

For now, manually assign Pro Member roles by asking users to DM you their account email. In the future, this can be automated via a Stripe webhook → Discord bot integration.

Set Permissions

Lock down the right channels so the structure works properly.

  • A
    Make Welcome channels read-only:
    Click any Welcome channel → Edit Channel → Permissions → @everyone → set Send Messages = ❌ and Add Reactions = ❌
  • B
    Make #changelog announcements-only:
    Edit Channel → Channel Type → switch to Announcement Channel. Only Admins can post, everyone else sees it as read-only.
  • C
    Gate the Pro Only category:
    Right-click 💎 PRO ONLY category → Edit Category → Permissions
    → @everyone: set View Channel = ❌
    → Click + Add Role → add ⭐ Pro Member → set View Channel = ✅
    → Click + Add Role → add 🛡️ Admin → set View Channel = ✅
  • D
    Sync all channels in Pro Only category:
    For each channel inside 💎 PRO ONLY → Edit Channel → Permissions → click "Sync with Category"

Add Bots

Two bots handle auto-welcome messages, moderation, and reaction roles. Both are free.

🤖

MEE6 Free

Handles welcome DMs, auto-moderation, reaction roles, and level/XP system. The most popular Discord bot.

  1. Go to mee6.xyz → click Add to Discord → select Bloggy Community
  2. Enable Welcome plugin → set DM message:
    "Hey {user}, welcome to Bloggy! Head to #start-here to get going, and post in #get-help if you need anything. Claim your early access deal here: https://bloggy.online/#early-access 🚀"
  3. Enable Reaction Roles → post a message in #welcome: "React 🌱 to get your Free Member role" → link 🌱 to the Free Member role
  4. Enable Moderator → turn on anti-spam and anti-links (prevents spam bots)
  5. Optional: enable Levels — members earn XP for chatting, creates engagement
📋

Ticket Tool Free

Turns support requests into private threads. Cleaner than public channels for sensitive issues.

  1. Go to tickettool.xyz → Add to Server
  2. Create a ticket panel in #get-help: users click a button → opens a private thread with you and the user only
  3. Set auto-close: "Close ticket after 48 hours of inactivity"
  4. Set close message: "Thanks for reaching out! If this helped, drop a ⭐ review on G2 or Product Hunt 🙏"

⚡ Optional Power-Up

Carl-bot — more powerful than MEE6 for reaction roles and auto-moderation. Free tier is generous. Use it if MEE6 feels limiting. carlbot.io

Use It as a Sales Tool

Discord isn't just support — it's your best conversion engine. Here's how to work it.

🏆

Post Wins Constantly

Every time Boom Media publishes a good post or a client gets a result — screenshot it and post in #wins. Real proof converts better than any ad copy.

📢

Announce Every Feature

Post in #changelog whenever something ships. Free users watching new features get FOMO → they upgrade. Keep them feeling the product is alive.

💎

Make Pro Feel Exclusive

Occasionally hint at what's being discussed in #pro-lounge or #agency-strategies. The exclusivity of a locked channel is a real upgrade driver.

⚡

Pin the Early Access Deal

Pin the EARLY50 offer in #welcome and #general. Mention it in your welcome DM. Every new member should see it within 30 seconds of joining.

🤝

Answer Fast

Respond to every #get-help post within a few hours. Fast support builds trust and word-of-mouth. It's the #1 thing that turns free users into paid.

🎙️

Run Monthly Q&As

Once a month, jump into a Discord Voice channel for a live "Office Hours" session. Answer questions, show new features, build personal connection with members.

📈

Share the Roadmap

Post upcoming features in #roadmap. Let members vote on what to build next. This drives engagement and keeps users subscribed waiting for features they care about.

🔗

Link from Bloggy App

Add a Discord button inside the Bloggy dashboard and on the landing page. Every user who hits a problem should have one click to reach support.

💰 The Conversion Flow

  • Visitor joins Discord → sees #welcome with early access deal → signs up free
  • Free user asks question in #get-help → you answer fast → builds trust
  • They see #wins posts from Pro members → want results like that
  • They see #changelog and #roadmap → product feels alive and worth paying for
  • They upgrade → DM you for Pro Member role → enter the exclusive channels
  • They feel part of something → refer other agency owners → viral growth
